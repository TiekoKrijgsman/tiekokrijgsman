# Vim (Vi IMproved)

Vim is a free and open-source, screen-based text editor program.

```sh
vim <file>
```

## Commands

All commands are preceded by `:`, and can be combined, e.g. `:wq` to save (`w`) and quit (`q`).

- `i`: insert mode
- `q`: quit
- `q!`: quit without saving (`!` forces)
- `:w`: save
- `%d`: delete all lines
- `yy`: copy all lines
- `p`: paste
- `u`: undo
